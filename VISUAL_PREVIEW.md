## Interaktivní prvky

### Hover efekty
```
Karty obsahu:
Normal:  ┌─────────────┐
         │  Text       │
         └─────────────┘

Hover:   ┌─────────────┐  <- Zvedne se o 5px
         │  Text       │     Border změní barvu
         └─────────────┘     Fialová záře (box-shadow)
```

### Tlačítka
```
Normal:  ┌────────────────┐
         │  Zjistit více  │  <- Fialové pozadí
         └────────────────┘

Hover:   ┌────────────────┐
         │  Zjistit více  │  <- Průhledné, fialový border
         └────────────────┘     Zvedne se o 2px
```

### Navigace
```
Normal:  O konceptu  Kritéria  Katalog

Hover:   O konceptu  Kritéria  Katalog
         ──────────              <- Fialová čára
```

## Animace

- **Fade in při scrollování** - karty se postupně objevují
- **Smooth scroll** - plynulé scrollování k sekcím
- **Hover animace** - plynulé přechody (0.3s)

```
Nadpisy:

Text:
```

## Ikony

```
✓ - Splněná kritéria (zelená)
❌ - Co to není (červená)
📄 - Odkaz na dokument
🔗 - DOI link
📚 - Prázdný katalog
☰ - Mobilní menu (pro budoucnost)
```
